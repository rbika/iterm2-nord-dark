# iTerm2 - Nord Dark

A darker version of [Nord](https://www.nordtheme.com/) theme for iTerm2.

![](screenshot.png)

## Install

- Right-click [`Nord Dark.itermcolors`](https://raw.githubusercontent.com/rbika/iterm2-nord-dark/master/Nord%20Dark.itermcolors) and choose `Save Link Asâ€¦`.
- Double-click the downloaded file.
